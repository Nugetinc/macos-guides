installing

1: requirements

  • vmware workstation 17
  • macos 14 iso
  • vmware unlocker for macos
  • At least 60GB disk space and 8GB+ RAM

2: unlock vmware for macos

  1. download auto-Unlocker by Paolo.
  2. extract and run win-install.cmd as administrator.
  3. This adds the macos option in vmare.

3: create a new virtual machine

  1. Open vmware and click Create a New Virtual Machine.
  2. select Typical (recommended).
  3. choose I will install the OS later.
  4. Select Apple Mac OS and version macOS 14
  5. name it and choose a location.
  6. allocate at least 60GB virtual disk.
  7. finish then click edit virtual machine settings to:
    • Set ram: 8–16GB
    • processors: 2 or more
    • Attach the macOS 14 ISO to the CD/DVD drive

Step 4: Add macOS EFI Tweaks

Edit the VM’s .vmx file and add the following:

smc.version = "0"
cpuid.0.eax = "0000:0000:0000:0000:0000:0000:0000:1011"
cpuid.0.ebx = "0111:0101:0110:1110:0110:0101:0100:0111"
cpuid.0.ecx = "0110:1100:0110:0101:0111:0100:0110:1110"
cpuid.0.edx = "0100:1001:0110:0101:0110:1110:0110:1001"
cpuid.1.eax = "0000:0000:0000:0001:0000:0110:0111:0001"
cpuid.1.ebx = "0000:0010:0000:0001:0000:1000:0000:0000"
cpuid.1.ecx = "1000:0010:1001:1000:0010:0010:0000:0011"
cpuid.1.edx = "0000:0111:1000:1011:1111:1011:1111:1111"
board-id = "Mac-937A206F2EE63C01"
hw.model.reflectHost = "FALSE"
hw.model = "MacBookPro15,1"

5: Start the vm and install macOS

  1. power on the vm
  2. format via disk util
  3. install macos sonoma normally from the installer.

    4. THIS CAN AND MOST LIKELY WILL TAKE A WHILE (UPWARDS OF 2 HOURS)

6: post-install tweaks

  • install vmware tools for resolution and input sync.
